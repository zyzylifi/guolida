# guolida
bootstrap+JQ+node.js+express商城项目，带后台MySQL数据库
### 此项目为node开发项目，数据库使用MySQL

1. 把guolida.sql文件放到数据库中执行
2. 所有的npm包都已上传，不用重新下载，如果要重新下载，则依据package.json文件
3. 数据库连接位于utils文件下，DBHelper.js中，需要更改链接地址和密码等
4. 运行在终端使用`npm start`
